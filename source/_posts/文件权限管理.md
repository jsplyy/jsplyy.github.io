#### 改变单个文件权限

```
chmod 777 filename
```


#### 目录权限说明
```
对于目录来说，执行位的作用是控制能否进入或者通过该目录，而不是控制能否列出它的内容。读取位和执行位的组合的作用才是控制是否列出目录中的内容。写入位和执行位的组合则是允许在目录中创建，删除，和重命名文件。
```

#### 递归改变目录及目录文件权限
```
chmod -R dirname
```

#### 递归改变文件属主
```
chown -R dirname
```
